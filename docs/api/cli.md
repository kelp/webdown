# Command-Line Interface

This page documents the Command-Line Interface (CLI) for Webdown,
which allows you to convert web pages to Markdown from the terminal.

## Main Entry Point

::: webdown.cli.main
    options:
      show_signature: true
      show_source: true

## CLI Argument Parsing

::: webdown.cli.parse_args
    options:
      show_signature: true
      show_source: true
