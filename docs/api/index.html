Webdown API Documentation

Welcome to the Webdown API documentation. Webdown is a Python tool for converting web pages to markdown. This documentation provides details about the modules, classes, and functions available in the Webdown package.

Modules

webdown

The main package module with version information.

webdown.cli

Command-line interface for webdown. Provides the entry point for the command-line tool.

Key functions:

  • parse_args() - Parse command line arguments
  • main() - Main entry point for the CLI

webdown.converter

Core functionality for HTML to Markdown conversion.

Key functions:

  • validate_url() - Validate URL format
  • fetch_url() - Fetch HTML content from URL
  • html_to_markdown() - Convert HTML to Markdown
  • convert_url_to_markdown() - End-to-end conversion from URL to Markdown

Exceptions:

  • WebdownError - Base exception class
  • NetworkError - Exception raised for network-related errors
  • InvalidURLError - Exception raised for invalid URL format

Usage Examples

Basic usage as a command-line tool:

webdown https://example.com
webdown https://example.com -o output.md
webdown https://example.com -c -t

Basic usage as a Python library:

from webdown.converter import convert_url_to_markdown

# Simple conversion
markdown = convert_url_to_markdown('https://example.com')

# With options
markdown = convert_url_to_markdown(
    'https://example.com',
    include_toc=True,
    include_links=True,
    include_images=False,
    css_selector='main',
    compact_output=True
)
